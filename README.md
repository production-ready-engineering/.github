# Stop Coding. Start Engineering.

_A 3-Month engineering program - Production Ready: Engineering Forge._

## The Problem 

You finished your studies or are thinking of landing a job already, but "Junior" roles require 1–2 years of experience. Why? Because companies aren't looking for people who just know syntax - they need engineers who can solve problems, navigate complex systems, and work within a professional team.

## The Reality Check

In the professional world, writing code is only one part of the job. The real challenge lies in architecture, systems thinking, and operational stability. **We bridge the gap between "knowing how to code" and "knowing how to deliver value."**

## What You’ll work on

- **Engineering vs. Coding**: Design solutions on paper first. Master MVP principles and architectural diagrams before writing a single line.  
- **Real-World Operations**: Gain the "production readiness" schools don't teach—logging, monitoring, and troubleshooting breaking systems.  
- **Team Dynamics**: Experience the reality of professional collaboration: Agile Sprints, peer code reviews, and collective accountability.  
- **AI-Assisted Engineering**: Learn to use AI as a tool for implementation while you retain full control over the system design and problem-solving logic.

## The 3-Month Journey

The program is structured to simulate a real engineering team’s workflow, moving from foundational principles to intensive, collaborative project work.

1. **Introduction and Assessment (Initial Week)**: The program starts with an introduction to our goals and expectations, followed by a First Interview to assess your current soft skills and technical readiness, providing you with immediate, personalized feedback.
2. **Foundational Principles (Approx. 2 Weeks)**: Intensive sessions will cover core engineering concepts, including learning how to learn effectively, advanced Googling, efficient prompt writing, and key work types such as REST API design and Background Workers (using tools like Rabbit MQ and Hangfire). This phase concludes with four individual tasks to apply the theory.
3. **Group Project Simulation (Approx. 2.5 Months)**: The majority of the program is dedicated to working on a real group project in Agile Sprints (Scrum vs. Kanban). You will conduct requirements analysis, perform solution design (including creating diagrams), and prepare the entire environment (skeleton code, Docker setup, CI pipeline, and testing framework).
4. **Career Readiness and Final Evaluation**: Throughout the program, you will develop soft skills, including professional communication, providing feedback, and task prioritization. The program culminates in training on CV/Interview best practices and a final, real-life interview simulation, including both technical and soft skills checks to ensure you are market-ready.
